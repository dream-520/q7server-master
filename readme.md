使用教程: 大家可以搜索下怎么加载自定义的插件.

大体说下流程:
1. 下载zip包即可
2. 解压后, 打开以下链接 chrome://extensions/
3. 打开右上角的开发模式, 然后加载你下载的这个文件夹就可以了~
